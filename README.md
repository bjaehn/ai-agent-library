# AI Agent Library

A structured library of reusable AI agent prompts for strategy, product management, research, execution, and software delivery.

This repository is designed to help product leaders, founders, strategists, and builders run repeatable, evidence-grounded analysis using specialized AI agents.

## Core Principle

> Evidence first. Framework second. Narrative last.

Agents should separate verified facts, assumptions, interpretation, and recommendations.

## Recommended Agent File Format

Each agent file should follow this structure:

```markdown
# Agent Name

## Role
## Purpose
## When To Use
## Inputs Required
## Operating Principles
## Process
## Output Format
## Quality Bar
## Guardrails
## Prompt
```

This repository is a living library. Agents should evolve as they are tested in real workflows and improved based on output quality, reliability, and usefulness.
